# Deep research skill links

This documentation points to two complementary research skill repositories:

- `x-research-skill`: X/Twitter research agent for Claude Code and OpenClaw, providing CLI search/filter/monitor workflows and citation-ready summaries.
  https://github.com/rohunvora/x-research-skill

- `claude-deep-research-skill`: full Claude Code deep research engine with citation-backed reports, multi-provider retrieval, validation loops, and strong quality standards.
  https://github.com/199-biotechnologies/claude-deep-research-skill

- `deep-research-skills` (Weizhena): structured deep research workflow for Claude Code/OpenCode/Codex with two-phase outline + investigation mode, multi-language support, and explicit web search agent integration.
  https://github.com/Weizhena/deep-research-skills
